# RegexHub

A quick reference cheatsheet for regular expressions.

## Features

- Anchors, character classes, quantifiers
- Groups, lookarounds, flags
- Escape sequences, substitution
- Common patterns for email, URL, IP, dates
- JavaScript and Python specific syntax
- POSIX character classes
